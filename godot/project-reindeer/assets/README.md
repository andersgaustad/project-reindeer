# Assets

Put `dev_assets.zip` here and extract it to a directory with the name `dev_assets`.
Afterwards, the project expects the aforementioned directory `dev_assets` to exists in the same directory as this README.

All assets are ignored by the [.gitignore](.gitignore), but extracting `dev_assets` should provide the project with all of the required assets and resources.
Asset packs that are used as-is should be but in [source](dev_assets/source), while assets that are derived, edited, or otherwise manipulated should be put in [dervied](dev_assets/derived/).

See the table below for asset packs and associated sources.
